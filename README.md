# StayBold

A mobile-first web app for tracking daily health habits.

## Overview

StayBold helps you monitor three key daily habits:
- **Alcohol consumption** (6 levels: no drinks, few drinks, tipsy, drunk, hammered, black out)
- **Exercise** (yes/no)
- **Drug use** (yes/no)

## Features

- Monthly calendar view optimized for mobile
- Simple tap-to-log interface with modal inputs
- Navigate backward through previous months
- Edit any previously logged day
- Typographic-focused design

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python Flask
- **Data Storage:** JSON file

## Project Status

In development. Authentication and analytics features planned for future releases.
